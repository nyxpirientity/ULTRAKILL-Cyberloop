# Important!
For this plugin/mod to work, you'll need to ensure your BepInEx is configured to have *`HideManagerGameObject`* set to *`true`*, or else the game will destroy the plugin!

# About
Cyberloop is a mod/cheat that semi-jankily adds portal layouts to cybergrind, there are about 8 layouts currently, some of which are off by default and can be toggled on in the configuration file. These layouts generally make the cybergrind arena loop, hence the name. 

# Performance Notice
If you've played any ULTRAKILL level that makes heavy use of portals, there's a good chance you've noticed that the portals are resource intensive. This mod uses the same portals as the game does, so in general it causes a pretty hefty hit to performance. I've been trying to come up with a way to handle this but currently it still seems like the answer is to just wait until they optimize them more, some how.
